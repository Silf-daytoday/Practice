Описание проекта

Проект направлен на развитие доступности цифровых сервисов Московского Политехнического Университета для его студентов и сотрудников.

Основные функции системы:

  • Сбор данных с датчиков оборудования в реальном времени
  • Анализ показателей с использованием алгоритмов машинного обучения
  • Прогнозирование возможных отказов оборудования
  • Формирование рекомендаций по техническому обслуживанию
  • Визуализация данных для операторов

Технологический стек:

Python

TensorFlow

IoT платформа
